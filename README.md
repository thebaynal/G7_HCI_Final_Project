# G7 HCI Final Project

This repository contains a simple Bootstrap starter template for the G7 Human-Computer Interaction final project.

To preview locally:

1. Open `index.html` in a browser, or
2. Serve with a simple static server (e.g., `npx serve` or `python -m http.server 8000`).

Files added:
- `index.html` — main template
- `css/style.css` — minimal custom styles

Pages in this site:
- `index.html` — Homepage
- `about.html` — About us page
- `news.html` — News and announcements

View locally (quick):

```powershell
# from the repo folder
python -m http.server 8000
# then open http://localhost:8000/index.html
```

Or open `index.html` directly in your browser (works for static demo).
